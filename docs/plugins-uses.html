Plugin's Usage Information

We are using multiple awesome plugins (optional) to build out the application pages and others. The following table is describing the necessary assets to include in your *.blade.php file if you are looking to use the same.

You can add the required javascript file inside block @section('script') and it would be available in your page. Similarly you can the required css file inside block @section('css').
